Welcome to FetchFox, a scraping API that lets you **scrape millions of items** with just **one line of code**.

FetchFox magically handles all the complexity and pain of scraping, including:

- **Automatic HTML parsing** using AI models
- **Proxy selection** to make sure you can access any page
- **Cost minimization** by reducing context for AI and managing proxy usage
